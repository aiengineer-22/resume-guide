# 📘 ATS Resume Guide

## 🔍 What is ATS?

ATS (Applicant Tracking System) is software used by companies to filter resumes before a human sees them.

👉 If your resume is not ATS-friendly, it may get rejected automatically.

---

## ❌ Common Mistakes

* Using fancy designs or graphics
* Adding images or icons
* Writing long paragraphs
* Not using keywords from job description

---

## ✅ Best Practices

### 1. Keep it Simple

* Use basic fonts (Arial, Calibri)
* Avoid tables and images

### 2. Use Keywords

* Read the job description carefully
* Include relevant skills and tools

### 3. Strong Bullet Points

❌ Weak:

* Worked on a project

✅ Strong:

* Analyzed 10,000+ rows of sales data using Python (Pandas) and identified trends that increased revenue by 15%

---

### 4. Use Proper Sections

* Contact Information
* Skills
* Education
* Projects / Experience

---

## 📌 Golden Rule

👉 Show IMPACT, not just tasks.

Example:

* ❌ Built a machine learning model
* ✅ Built a classification model using Scikit-learn with 92% accuracy to predict customer churn

---

## 🎯 Final Tip

Your resume should be:
✔ Clear
✔ Relevant
✔ Results-driven
